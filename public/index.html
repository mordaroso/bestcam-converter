How to embed

Sample HTML to paste into your blog sidebar: 
  <div id="bestcam-badge" data-id="{YOUR PHOTOGRAPHER ID}"></div>
  <script src="http://bestcam.fabiokuhn.com/js/bestcam-badge.js" type="text/javascript"></script>
Important: You have to set the photographer id which you can find on The Best Camera site.

Get the code

The Best Camera Badge is open source and hosted on GitHub
  • The Javascript and Stylesheets: http://github.com/mordaroso/bestcam-badge
  • The simple Rack application needed for the data: http://github.com/mordaroso/bestcam-converter
    (because there is no API provided by bestc.am)
